# slr-toolkit

A Toolkit for Systematic Literature Reviews

###Build
* cd repositories/de.tudresden.slr.thirdparty && mvn p2:site
* cd ../../ mvn clean package

### Dependencies
* JBibTex (BSD like license)
* iText (AGPL)
